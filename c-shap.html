១/ សេចក្ដីផ្ដើមនៃភាសា C#


C# (អានថា C Sharp)គឺជា Programming Language ដ៏ពេញនិយមមួយ ដែលត្រូវបាន Develop ឡើងដោយក្រុមហ៊ុន Microsoft (ដឹកនាំដោយលោក Anders Hejlsberg) ដែលដំណើរការនៅលើ .Net Framework។
.Net Framework គឺជា Class library ដ៏ធំមួយដែលមានផ្ទុកនូវ Class ជាច្រើនដើម្បីជាជំនួយទៅដល់ការសរសេរជាមួយ​ .Net Language ដូចជា C# ជាដើម។


១.១/ តើ C# អាចធ្វើអ្វីបានខ្លះ?


C# អាចត្រូវបានគេប្រើប្រាស់ទៅតាមគោលបំណងផ្សេងៗគ្នាដូចខាងក្រោម៖ -អាចយកមកប្រើបង្កើតជា Web Application
-អាចយកមកប្រើបង្កើតជា Desktop Application
-អាចយកមកប្រើបង្កើតជា Mobile Application
-ធ្វើជា Web Service Application
-អាចយកមកធ្វើជា Game (PC, Mobile Phone) និងផ្សេងៗជាច្រើនទៀត។

១.៣/ Syntax នៃភាសា C#

using System;
namespace HelloWorld{
    class Program{
        static void Main(string[ ] args){
          //កន្លែងសម្រាប់សរសេរកូដក្នុង main method​
        }
    }
}

-using System:បានន័យថាយើងបានហៅ manespace ដែលមានឈ្មោះថា System យកមកប្រើប្រាស់ដែលមានផ្ទុកនូវ class ជាច្រើន។
-namespace: វាប្រៀបបានជា container មួយដែលគ្រប់គ្រងកូដទាំងមូលនៅក្នុង project ដែលមានឈ្មោះថា HelloWorld។
-static void Main(): គឺដូចគ្នាទៅនឹង main function ក្នុងភាសា C/C++ ដែល។

៣/ អថេរ (Variables) ក្នុងភាសា​ C#


Variable គឺប្រៀបបានជា Container មួយដែលត្រូវបានគេប្រើប្រាស់ក្នុងគោលបំណងដើម្បីរក្សាទុកទិន្នន័យផ្សេងៗ។នៅក្នុងភាសា C# មានប្រភេទ Variable ជាច្រើនដែលត្រូវបានកំណត់ដោយ keyword ដូចខាងក្រោម៖

-int: អាចឲ variable ផ្ទុកទិន្នន័យជាចំនួនគត់(integer)
-double: អាចឲ variable ផ្ទុកទិន្នន័យជាចំនួនទសភាគ​
-char: អាចឲ​ variable ផ្ទុកទិន្នន័យជាអក្សរមួយតួរ
-string: អាចឲ variable ផ្ទុកទិន្នន័យជាពាក់្យ ឬឃ្លា
-bool: អាចឲ​ variable ផ្ទុកតម្លៃតែពីរប៉ុណ្ណោះគឺ ពិត(True) និងមិនពិត(False)
-និងមានផ្សេងៗទៀតដូចជា៖ short, byte, float,...។

៣.២/ Type Casting


Type Casting គឺជាការបម្លែង(convert)ពីទិន្នន័យនៃប្រភេទមួយឲទៅជាប្រភេទមួយផ្សេងទៀត។នៅក្នុងភាសា C# គែបែងចែកការ convert នេះជាពីរគឺ implicit casting និងexplicit casting។


-Implicit Casting: គឺជាការ Convert ពីប្រភេទទិន្នន័យដែលមានទំហំតូចទៅជាប្រភេទទិន្នន័យដែលមានទំហំធំ។ការ Conver នេះគឺមានលក្ខណះជាស្វ័យប្រវត្តិ។

#ឧទាហរណ៍៖​int num1=10;
double num2=num1;
//num1 ត្រូវបាន convert ទៅជាប្រភេទ double

-Explicit Casting: គឺជាការ Convert ពីប្រភេទទិន្នន័យដែលមានទំហំធំទៅជាប្រភេទទិន្នន័យដែលមានទំហំតូច។ការ Conver នេះគឺតម្រូវឲអ្នកបញ្ជាក់ពីប្រភេទដែលអ្នកចង់ Convert ឲបានច្បាស់លាស់។

#ឧទាហរណ៍៖​double num1=10;
int num2=(int)num1;
//num1 ត្រូវបាន convert ទៅជាប្រភេទ int


នៅក្នុង​ emplicit casting នេះ C# ក៏បានផ្ដល់ឲអ្នកអាចប្រើប្រាស់នូវ Method មួយចំនួនសម្រាប់ការ​ Convert នេះបានផងដែរ។

-Convert.ToBoolean(): ការ convert ទៅជា bool
-Convert.ToDouble(): ការ convert ទៅជា double
-Convert.ToInt32(): ការ convert ទៅជា int
-Convert.ToInt64()): ការ convert ទៅជា long
-Convert.ToString(): ការ convert ទៅជា string

#ឧទាហរណ៍៖​ ខាងក្រោមនេះជាការ convert ពី int ទៅជា string

៦/ Control Statement ក្នុង C#


នៅក្នុងមេរៀន Control Statement របស់ C# នេះគឺមានលក្ខណះស្មើរតែដូចគ្នាទាំងស្រុងទៅនឹង Control Statement នៅក្នុងភាសា C/C# ផងដែរ។ដូច្នេះ Admin មិនបានលម្អិត ឬពន្យល់ឲបានគ្រប់ជ្រុងជ្រោយនោះឡើយ។


៦.១/ If Else Contition


Syntax:
if(condition-expression){
      //statement 1 will work if boolean if "true"
}else{
      //statement 2 will work if boolean if "false"
}

៦.២/ Switch Statement


Syntax:
switch(expression){
    case value1:
                   //statement1
                   //break;
    case value2:
                   //statement2
                   break;
    default:
               //this block runs when no case matches
}

៦.៣/ Loop Statement


Syntax: (for loop)
for(initialization;expression;iterator){
    //loop statement
}

-initialization:សំដៅទៅលើការកំណត់តម្លៃចាប់ផ្ដើម
-expression:ជាការកំណត់នូវលក្ខខណ្ឌទៅលើ loop
-iterator:ជា​​ increment ឬdecrement


៦.៤/ Foreach Loop Statement


Foreach Loop ត្រូវបានគេប្រើប្រាស់ដើម្បីធ្វើការ​ loop ទៅលើ Element (ធាតុ) របស់ Array.

Syntax:
foreach(type var_Name in array/collection){
    //Code block to be executed
}

Arrays គឺជាបណ្តុំ (collection) ទិន្នន័យជាច្រើន ដែលមានប្រភេទទិន្នន័យ (data type) ដូចគ្នា។នៅក្នុងមេរៀននេះយើងនឹងសិក្សាអំពីរបៀបនៃការបង្កើត, ការ initialize និងការចូល access ធាតុរបស់​ array។


៧.១/ ការបង្កើត Array

Syntax:
          dataType[ ] array_name;

        string[ ] name;
        double[ ] salary;

៨/ Array ច្រើនវិមាត្រ


ចំពោះ array ច្រើនវិមាត្រ(multidimension arrays) នៅក្នុងភាសា C# គេច្រើនតែសិក្សាទៅលើ array ២វិមាត្រ ឬត្រឹម៣វិមាត្រប៉ុណ្ណោះ។ ទិន្នន័យនឹងត្រូវបានផ្ទុកជាតារាងដែលមានជួរដេក(row) និងជួរឈរ(column) ដែលត្រូវបានគេហៅថា ម៉ាទ្រីស(martix)។

Function គឺជាសំណុំ/block កូដដែលត្រូវបានបង្កើតឡើងដើម្បីដំណើរការរួមគ្នានៃការងារជាក់លាក់ណាមួយ។ នៅក្នុង​ function នៃភាសា C# មាននូវ component សំខាន់ៗដូចជា៖
-Function Name: ឈ្មោះរបស់ function។
-Return Type: ត្រូវបានកំណត់ប្រភេទទិន្នន័យជាក់លាក់ពេលដែលប្រើប្រាស់ជាមួយ Returned-function។
-Body of Function: ជាកន្លែងដែលត្រូវសរសេរកូដផ្សេងៗដែលទីតាំងស្ថិតក្នុងសញ្ញា braces({})។
-Access-Specifier: ជាការកំណត់កម្រិតនៃការ Access ក្នុងករណីដែលមានការ​ access ពីខាងក្រៅ class ជាដើម។
-Parameters: ជាបណ្ដុំនៃ argument ផ្សេងៗដែលត្រូវបានប្រើនៅក្នុងការផ្ទេរទិន្នន័យពី function មួយទៅ function ផ្សេងទៀត។

៩.២/ Function Call by Reference


នៅក្នុងភាសា C# មាននូវ keyword មួយគឺ ref ដែលត្រូវបានប្រើប្រាស់វា​ ដើម្បីកំណត់ទៅលើ argument ឲក្លាយទៅជាប្រភេទ reference។បានន័យថា argument ដែលជាប្រភេទ reference នឹងទទួលយក address របស់ argument ផ្សេងទៀតដោយមិន Copy Valu ដូចដែល Call by Value នោះឡើយ។

        }
        static void Main(string[ ] args){
               int num = 10,
               Console.WriteLIne("Old Num ="+ num);
               newNum(ref num);
          
               Console.WriteLine("New Num = " + num);
               Console.Readkey();
        }
    }
}

New Num = 20

Inside newNum(): 20
Outside newNum(): 10
