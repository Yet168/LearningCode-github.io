១/ ការណែនាំអំពី CSS

១.១/ អ្វីទៅជា CSS


ពាក្យពេញរបស់ CSS គឺ Cascading Style Sheets ប្រើប្រាស់សម្រាប់កំណត់ Style(color,layout,font...ect) អោយ Web page ដើម្បីមើលកាន់តែទាក់ទាញ នឹងស្រស់ស្អាត។

១.២/ CSS Syntax


CSS rule ត្រូវមាន selector នឹង declaration block

p {
      color: red;
      font-size: 20px;
}

* p ជា selector នៅក្នុង CSS.វា​ Point ទៅ HTML Element <p> ដែលយើងចង់កំណត់ style អោយ
* Declaration block អាចមានមួយឬច្រើន declaration ដែល separate ដោយ semicolon ();
* Declaration នីមួយៗមានផ្ទុកឈ្មោះ CSS Property នឹង value ដែល separate ដោយ​ colon (:)
* color ជា CSS property។ red ជា property value
* font-size ជា CSS property។20px ជា property value
លទ្ទផលនៅក្នុងឧទាហរណ៍ខាងលើគឺ៖​ គ្រប់ <p> Element មានពណ៍ក្រហម និងទំហំអក្សរ 20pixel។


២/ CSS Selector
២.១/ អត្តន័យ


CSS Selector ប្រើប្រាស់សម្រាប់រើសយក HTML Element(s) ណាមួយដែលអ្នកចង់កំណត់ style អោយវា។


២.២/ ប្រភេទ CSS Selector ចែកចេញជា៥៖


* Simple selector: Select element អាស្រ័យលើ name, id, class
* Combinator selector: Select element អាស្រ័យលើ specific relationship between them
* Pseudo-classselectors: select element based on a certain state
* Pseudo-element selectors: select and style a part of an element
* Attribute selectors: select element based on an attribute or attribute value
