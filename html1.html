១/ តើ HTML គឺជាអ្វី?


HTML គឺជាភាសាសម្រាប់បង្កើតគេហទំព័រ(Website)។HTML មកពីពាក្យ Hyper Text Markup Languageវាមិនមែនជាភាសា Programming Language ទេ តែវាជាភាសា Markup Language។ Markup Language គឺជាសំនំុនៃ Markup Tages ហើយដែល HTML ប្រើប្រាស់ markup tags ដើម្បីកសាងគេហទំព័រ ហើយធ្វើការបង្ហាញដោយ Web Browser។ Web page កើតឡើងដោយទំនាក់ទំនងជាច្រើន Link ទៅវិញទៅមកសម្រាប់បង្ហាញ Contents។ HTML ជាភាសាមួយដែលងាយស្រួលក្នុងការសិស្សប្រើប្រាស់។

<tag syntax: name= "value">...............................................................................................</tag>
